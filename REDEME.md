# hypr
> 安装

```bash
git clone --recursive https://github.com/hyprwm/Hyprland
cd Hyprland
make all && sudo make install
```

> 更新

```bash
git pull
make all && sudo make install
```

> 更新子模块

```bash
git submodule update --remote --merge
make all && sudo make install
```
